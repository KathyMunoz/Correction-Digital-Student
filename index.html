Codez des sites web modernes avec Html et Css.

Graçe au language Html et Css vous pouvez structurer et créer des sites web de cero.

Pour quoi apprendre HTML/CSS?

Si vous vous demandez les raisons pour lesquelles il faut apprendre à coder en Html/Css, voilà pour quoi:

  • Meilleure structure de votre site web avec html.
  • Creativité infinite avec css et ses créations d'animations.
  • On peut construire des sites web simples avec des balises de base, puis progresser vers des projets plus complexes.
  • Compatibilité avec d'autres tecnologies.

Les balises de base

Les balises autour du contenu permettent de transformer une image ou un texte pour obtenir un lien vers une autre page, créer des titres,etc.

  • Les titres avec <h1>, <h2> … 
    <h1>Mon titre</h1>
  • Le paragraphe avec <p> 
    <p>Ceci est un paragraphe</p>
  • Les listes avec <ul> et <li>
                                <ul>
                            <li>Tomates</li>
                            <li>Mozzarella</li>
                            <li>Basilic</li>
                            </ul>

Tips et bonnes pratiques

Un codage CSS bien structuré est cruciale pour la maintenabilité et la lisibilité du code, voilà quelques conseils:

  1. Organiser les styles par sections logiques
  2. Utiliser des commentaires pour expliquer les blocs de code
  3. Regrouper les propriétés similaires.
